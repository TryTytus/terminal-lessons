# git branch

Create, rename, list, and delete branches.

## Daily forms

- `git branch` - List local branches.
- `git branch feature/api` - Create branch.
- `git branch -m old new` - Rename branch.
- `git branch -d feature/api` - Delete merged branch.

## Example

```sh
git branch feature/search
```

## Practice note

Delete only branches that were already merged.
