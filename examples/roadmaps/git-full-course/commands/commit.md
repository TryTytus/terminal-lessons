# git commit

Record staged changes as a snapshot in history.

## Daily forms

- `git commit -m "message"` - Commit staged changes.
- `git commit --amend -m "new message"` - Rewrite latest commit message.
- `git commit --amend --no-edit` - Add staged changes to latest commit.

## Example

```sh
git commit -m "add login form"
```

## Practice note

Keep commit messages short, imperative, and specific.
