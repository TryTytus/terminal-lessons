# git log

Browse commit history in compact or detailed forms.

## Daily forms

- `git log --oneline` - One-line commits.
- `git log --graph --decorate --all` - Visual graph across branches.
- `git log -n 5` - Show the latest five commits.
- `git log --stat` - Show per-file change stats.

## Example

```sh
git log --oneline -n 10
```

## Practice note

Use `--graph --all` when branch relationships are unclear.
