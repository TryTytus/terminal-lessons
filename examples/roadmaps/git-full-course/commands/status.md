# git status

Inspect tracked, staged, unstaged, and untracked changes.

## Daily forms

- `git status` - Full human-readable output.
- `git status --short` - Compact two-letter status view.
- `git status -sb` - Short view with branch header.

## Example

```sh
git status --short
```

## Practice note

Read status before and after every `add`, `commit`, `restore`, or `reset`.
