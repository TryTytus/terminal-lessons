# wc

Count lines, words, and bytes with redirection for clean numeric output.

## Daily forms

- `wc -l <file>` - Count lines.
- `wc -w <file>` - Count words.
- `wc -c <file>` - Count bytes.

## Example

```sh
wc -l < names.txt > line-count.txt
```
